## 目录结构

```
lib                 // 类库
res                 // 资源
---font
---img
---music
src                 // 源码(MVC)
---control          // Control
------action        // 动作（页面跳转等）
------api           // 接口
------app           // 应用级别
------base          // 基本（基类）
------common        // 共同（待定...）
------global        // 全局（变量+字符串）
------static        // 静态（变量+字符串+资源+UI）
------util          // 帮助
---model            // Model
---view             // View
------componment    // 组件
------page          // 页面
```
